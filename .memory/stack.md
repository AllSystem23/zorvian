# Stack

## Proyecto
- **Nombre:** Zorvian ERP
- **Significado del nombre:**
  - **Zor:** fuerza, impulso y determinación para crecer
  - **Via:** camino, dirección y progreso
  - **An:** evolución continua y visión hacia el futuro

## Backend
- .NET 9 (ASP.NET Core, Minimal APIs)
- Entity Framework Core 9 + PostgreSQL (Npgsql)
- FluentValidation, Mapster
- xUnit + Moq + FluentAssertions
- Swagger / OpenAPI

## Frontend
- Flutter 3.x (Dart)
- Riverpod (estado)
- Dio (HTTP)
- Drift (base de datos local SQLite/web)
- SharedPreferences, SQFlite, Geolocator, Camera, CSV, Share Plus

## Infraestructura
- PostgreSQL (Neon)
- GitHub Actions (CI/CD)
- dotnet user-secrets (credenciales locales)
