# Stack

- **Backend:** .NET 9, Clean Architecture (Core/Application/Infrastructure/Web)
- **Frontend:** Flutter 3.12+, Riverpod 3.3, GoRouter 17, Material 3
- **Database:** PostgreSQL (Neon)
- **Auth:** Firebase Auth + JWT propio
- **ML:** ML.NET 4.0.0 (FastTreeRegression)
- **Jobs:** Hangfire (recurring: absentismo domingo 2AM, ventas domingo 3AM)
- **API pattern:** REST, route prefix `zorvian/v1/`
- **HTTP Client:** Dio (con interceptors auth + refresh)
- **Charts:** fl_chart
- **Design System:** ZTokens (ZColors, ZTypography, ZRadii, ZShadows) + ZComponentes (ZButton, ZTextField, ZSelect, ZBadge, ZCard, ZModal, ZToast, ZDataTable, ZStepper, ZTimeline, ZCalendar, ZSkipLink, ZMainContent, ZLiveRegion)
